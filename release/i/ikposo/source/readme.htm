ikposo

a keyboard using the qwerty layout for typing in the endangered ikposo language.
This keyboard layout is customized for typing in the Ikposo language. Use the following key mappings for special characters:
x → ɛ
c → ɔ
; → ʋ
j → Ɩ
q → ŋ
These keys allow accurate representation of Ikposo phonemes using standard keystrokes.
